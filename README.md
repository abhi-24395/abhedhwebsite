# abhedh.in

Static single-file site. No build step.

**Site:** abhedh.in
**Repo:** https://github.com/abhi-24395/abhedhwebsite

## Files

- `index.html` — the site
- `CNAME` — custom domain (`abhedh.in`)

## Deploy on GitHub Pages

1. Push to `main`.
2. **Settings → Pages** → Source: **Deploy from a branch** → Branch `main`, folder `/ (root)`.
3. Wire `abhedh.in` as a custom domain.

## Edit

Open `index.html`. No build, no server.
